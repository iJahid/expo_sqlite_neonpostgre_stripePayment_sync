# React Native expo app wsing Talwind Neon and Stripe and Auto Sync  👋

created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).
It include

1. Tailwind CSS
2. Google Authentication
3. Email and Password login/signup
4. SQL lite local database
5. Neon PostgreSQL serverless Online Database
6. Both way Sync
7. Strip Payment Methode for VISA/MASTER etc.
   
## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```
